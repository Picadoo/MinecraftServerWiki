# 🚀 新手快速入门

欢迎来到 **[幻想之境 RPG]**！为了让你能顺利踏上冒险之旅，请按照以下步骤操作。

## 第一步：连接服务器

我们的服务器支持 Minecraft Java 版的 **1.16.5** 及以上版本。

### 服务器地址 (IP)

> ```
> play.yourserverip.com
> ```
>
> _请确保您的游戏版本在 1.16.5 - 1.20.4 之间。_

## 第二步：进入游戏

连接服务器后，您会自动进入新手村 **“晨曦镇”**。

1.  **领取新手礼包：**
    在出生点找到 **新手导师 NPC** (头顶有 `[Guide]` 标记)，点击他领取您的启动装备和必需品。

    > **命令快捷方式:** `/kit starter`

2.  **选择职业：**
    您可以在导师附近找到 **职业石碑**。请仔细阅读每个职业的介绍，一旦选择，短时间内将无法更改！
3.  **了解基本 HUD：**
    - **生命值/饥饿值:** 显示您的基础状态。
    - **耐力条:** 在经验条位置显示，用于冲刺和施放部分技能。
    - **装备评分 (Gear Score):** 在聊天栏上方显示，这是进入高级副本的门槛。

## 第三步：第一次冒险

晨曦镇周边有一些简单的任务和区域供您练手。

### 推荐的起始任务

- **“清除田鼠”**：向农场主领取，可获得少量金币和经验。
- **“药剂采集”**：向炼金师领取，帮助您熟悉地图探索。
